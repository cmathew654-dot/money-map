# Money Map product brief

## Product

Money Map is an upscale, dynamic successor to the PowerPoint diagrams financial advisors use to explain a household's accounts, assets, income sources, reserves, needs, and relationships. It is a product tool with presentation-quality output, not a financial calculator.

## Primary user and job

The primary user is a financial advisor preparing and presenting a client's financial story. The product should help the advisor author a clear visual narrative quickly, then present it without editor chrome or implied analytical precision.

## Portfolio signal

The project must demonstrate WealthTech product judgment and frontend design-engineering taste equally. It should feel discerning, composed, editorial, and quietly confident.

## Financial truth boundary

- Every financial value is authored and displayed literally.
- The application does not calculate taxes, reconcile balances, infer funding capacity, debit accounts, annualize cash flows, or validate whether one amount can fund another.
- Geometry and styling never encode magnitude, confidence, capacity, or tax consequences.
- Approximation marks, ranges, blank placeholders, cadence phrases, and advisor-authored prose round-trip exactly.

## Public demo identity

- Brand: Cairn.
- Household: Hartwell family.
- All data is synthetic and illustrative.
- Real firm logos, client information, and raw reference screenshots are excluded from the public repository.

## Four equal-finish starters

1. Retirement Income — Private Ledger.
2. RMD & Withholding — Distribution Registry.
3. Annuity Income Floor — Foundation.
4. Roth Conversion — Conversion Path.

No starter is the hero or control fixture. Each receives a complete authored composition, editor, five-step presentation, responsive treatment, accessibility coverage, and final evidence.

## Non-goals for v0.1

No backend, authentication, cloud sync, collaboration, financial computation, arbitrary file import/export, auto-layout, freehand drawing, rich text, rotation, mobile authoring, or production-advisory claims.
